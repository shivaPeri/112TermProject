
READ ME


Project Name: Fourier Drawing Machine
Project Description: A program in which users can make a custom drawing machine to create spirograph drawings. The machine can be customized in the following ways: wheel radius/speed, number of wheels, pen color/size, etc. The user can save and view drawings at various playback speeds. The user can also upload an svg file and watch the generated drawing machine draw the path of the svg

Running the program:
run the file: termProject_v3.0
all necessary files are included

Libraries needed:
svg.path

In terminal,
pip3 install svg.path

Shortcuts:
use '<' and '>' keys to speed up or slow down drawing playback

Use '>' or '.' to speed up the drawing playback
Use '<' or ',' to slow down the drawing playback
